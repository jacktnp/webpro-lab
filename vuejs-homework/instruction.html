Simple Cineplex

ในการแก้ปัญหาต่างๆ ปัญหาหนึ่งๆ มักประกอบขึ้นมาจากปัญหาเล็กๆ จำนวนมากมารวมกัน - ปัญหาใหญ่แก้ยาก ปัญหาเล็กแก้ง่าย


หน้าแรก

1. จาก Wireframe ที่ให้มา ให้สร้าง Static HTML Page ตามที่ตัวเองพอใจด้วย HTML + CSS + Bootstrap (ในชั้นตอนนี้ยังไม่จำเป็นต้องมี JS)

2. import Vuejs เข้ามาในหน้าแรก และทดสอบว่า Vuejs ทำงานได้ถูกต้อง

3. Copy ข้อมูลจาก data.js มาไว้ใน data ของ Vue

4. ใช้ v-for แสดงหมวดหมู่หนัง (categories) ทั้งหมดตามข้อมูลใน data.js

5. ใช้ v-for แสดงภาพยนตร์ (movies) ทั้งหมดตามข้อมูลใน data.js

6. เขียนโค๊ดเพื่อ Filter ภาพยนตร์ตามหมวดหมู่

  • เพิ่ม v-on:click เข้าไปที่หมวดหมู่โดยให้ส่ง parameter เป็น id ของหมวดหมู่
  • สร้าง method ที่รองรับการทำงานของ v-on:click
  • ทดสอบการส่ง parameter ของ v-on:click (อาจทดลองโดย console.log() parameter ออกมาดู)
  • ใช้ Array.filter() ในการคัดกรองภาพยนตร์ตาม "id ของหมวดหมู่"

7. เขียนโค๊ดเพื่อ Search ภาพยนตร์

  • เพิ่ม v-on:keyup เข้าไปที่ Searchbox
  • สร้าง method ที่รองรับการทำงานของ v-on:keyup
  • ทดสอบ console.log() ข้อความที่อยู่ใน Searchbox
  • ใช้ Array.filter() ในการคัดกรองภาพยนตร์ตาม "ข้อความใน Searchbox"

8. ทำให้รูปโปสเตอร์ของภาพยนตร์เป็น link ไปที่หน้าแสดงรายละเอียดภาพยนตร์ -> ดู Homework 2
