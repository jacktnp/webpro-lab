โจทย์

  1. ทำ Exercise 3 โดยใช้ ModelForm
  2. ทำให้ปุ่ม สามารถลบ Choice ออกจากฐานข้อมูลได้
    • เพิ่ม path ใน polls/urls.py
    • เพิ่ม api view ใน polls/views.py
    • เมื่อปุ่ม ถูกกดให้แสดง Javascript Confirm Dialog ดังรูป
    • เมื่อกดปุ่ม "OK" ให้ส่ง request ไปลบ Choice ออกจากฐานข้อมูล
  3. ทำให้ /polls/1/add-choice/ สามารถแก้ไข Choice ได้
    1. แสดงรายการ choice ของ question ที่ถูกเลือก
      • สร้าง API ที่รับ <int:question_id> แล้ว return choice ในคำถามนั้นออกมาในรูปแบบ JSON (ตัวอย่าง)
      • ในหน้า /polls/1/add-choice/ ให้ใช้ axios ส่ง request ไปที่ API นำข้อมูลที่ได้จาก API มาแสดงในตาราง (ตัวอย่าง)
    2. ใน polls/views.py ให้แก้ไข add_choice_api ให้ทำการ
      • update เมื่อเป็น choice เก่า
      • create เมื่อเป็น choice ใหม่
